How to Create ICO File for Windows

  1. Click "Download All Sizes" above
  2. Go to icoconvert.com or convertico.com
  3. Upload the icon-256.png file
  4. Select sizes: 16, 32, 48, 64, 128, 256
  5. Download the ICO file
  6. Save as icon.ico in the desktop-app/assets/ folder
  7. Also save icon-256.png as icon.png in the same folder
